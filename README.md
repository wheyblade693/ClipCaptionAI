## 🚀 How to use the toolkit

You can use the software for several content tasks. Follow this guide to start your first project.

### Finding viral clips
Open the app and select the Clip Finder tool. Point the software to your long-form video file. The AI scans the footage for high-energy moments and pauses. It cuts these into shorter segments perfect for TikTok or YouTube Shorts. You can preview these clips and select the ones you want to keep.

### Adding captions
Once you select your clips, navigate to the Captions tab. The software transcribes your audio and places text on the screen. You can customize the font, color, and size of these captions to match your brand style. The software automatically times the text to appear when you speak.

### Inserting B-roll
The B-roll feature detects topics in your video. If you talk about a dog, the software suggests related stock clips to layer over your footage. You can turn this feature on or off in the settings menu. Place these clips in the timeline to make your video more engaging for your audience.

### Remotion workflows
For advanced users, the software supports Remotion workflows. You can export your project files into a format that allows for programmable video composition. Use this to create complex animations or bulk-produce content for different platforms.

## 🛠️ Frequently asked questions

Use this section to solve common setup issues.

**Does this software record my voice?**
No. You must provide existing video files. The software processes your files locally on your computer to protect your privacy.

**Why does the AI take time to process?**
Video editing requires a lot of math. The software analyzes every frame to place captions and perform scene detection. This takes time depending on the speed of your processor.

**Can I change the language of the captions?**
Yes. Head to the Settings menu and look for the language pack section. You can download and select different model files for specific languages.

**How do I update the software?**
When you open the app, it checks for updates. If a new version exists, the app prompts you to download the latest installer from the website. Run the new installer to overwrite the old files with the latest improvements.

## 📁 File structure and support

Your project files load into the main dashboard. You can save your work at any time using the Save Project button. This creates a small file with the extension .clip. You can reopen this file later to continue editing your video. 

If you encounter errors during the export process, ensure you have enough disk space. Large video files require significant temp space while the software stitches the final product together. Clear your cache in the settings tab if the software runs slow after long editing sessions.
